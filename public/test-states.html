WAICT Status Indicator Test

This page shows what the status indicator looks like in both states:

WAICT Enabled

Your Browser Supports WAICT

This is what you see when WAICT is enabled. In enforce mode, incorrect.js (which has a wrong hash) gets blocked. In report mode, violation reports are generated for mismatched hashes.

WAICT Disabled

Your browser does not support / has not enabled WAICT

This is what you see when WAICT is not enabled or not supported. All scripts load normally, including incorrect.js, and no violation reports are generated.

How it works

  • correct.js has the correct hash in the manifest → always loads
  • incorrect.js has an incorrect hash in the manifest → gets blocked only when WAICT is enabled
  • In enforce mode: WAICT is detected when incorrect.js is blocked
  • In report mode: WAICT is detected when violation reports are generated (resources still load, but violations are reported)
  • By default (WAICT off): incorrect.js executes normally and no reports are generated
